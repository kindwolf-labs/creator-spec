# Creator Spec

> *The first specification protocol written by a creator, for the AI age.*
>
> *Internally codenamed "Book of God" — a private name for a public protocol.*

---

## What is this?

If [OpenAI has Model Spec](https://model-spec.openai.com/) defining how
their models should behave, and Anthropic has Constitutional AI defining
their models' values — then **every serious creator needs their own Spec.**

**Creator Spec** is the first attempt at exactly that:
a faithful translation protocol between a human creator's complete intent 
and the machine intelligence executing it.

It is not a prompt framework.
It is not a workflow.
It is the missing layer between human ambiguity and machine precision.

---

## Why now?

The most underrated bottleneck of the AI era is not generation capability —
it is **the specification problem**.

- Models can generate 1,000 variations.
- Humans can't articulate which one they want.
- In complex creative domains (e.g. game development),
  intent-loss exceeds 80%.

Creator Spec aims to drive end-to-end intent-loss toward zero —
**for one creator, in one domain, first.**

---

## Origin

This protocol emerged from deep dialogue between a frontier LLM and 
a veteran creator — known online for over a decade as **"the kind wolf"** 
(善良的大灰狼) — who has lived through four media revolutions in 
Chinese gaming: PC → online → web → mobile.

Now entering the fifth, and the most important.

The protocol is being battle-tested in the development of an 
AI-native game, to be released in 2025–2026.

**The game is the proof. The Spec is the legacy.**

## A note on the codename

The internal name *"Book of God"* (上帝之书) is not a claim of divinity.
It is a creator's private term for a personal aspiration:
to write down, faithfully, what one means — so that no part of the 
intent is lost in translation.

Every creator, in their own studio, writes their own Book of God.
This repository is just the first one made public.
